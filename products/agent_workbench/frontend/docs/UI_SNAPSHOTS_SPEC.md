
# Fiido 智能工作台 & 计费门户 UI 视觉快照协议 (V1.0)

本文件作为视觉元数据，供后续具备 Vision 能力的 AI 识别并延续 UI 开发。

---

## 1. 全局设计规范 (Global Design System)
- **核心色盘**：
  - `Primary (Fiido Green)`: #00a6a0 (带 20% 透明度的发光阴影)
  - `Surface (Dark)`: #0f172a (用于工作台侧边栏、计费看板)
  - `Background`: #f8fafc (浅灰蓝，降低长时间用眼疲劳)
  - `Accent (Fiido Light)`: #f0f9f9 (用于激活态背景)
- **圆角规范**: 卡片采用 `rounded-[32px]` 或 `rounded-[40px]`，按钮采用 `rounded-xl`。
- **字体栈**: `Inter` (英文数字) + `Noto Sans SC` (中文)，标题加粗级别为 `font-black`。

---

## 2. 页面快照 A：独立计费门户 (BillingPortal.tsx)
**位置**：`components/BillingPortal.tsx`
**布局结构**：
1. **顶部 Header**: 
   - 左侧：Fiido 绿图标 + "Fiido 平台订阅与方案中心"。
   - 右侧：白色“消费明细”按钮 + 绿色“极速扩容”按钮。
2. **ROI 看板 (Hero Section)**:
   - **深色卡片**: 背景为 #0f172a。
   - **左侧**: 绿色大字显示“¥4,800”节省金额。
   - **右侧**: 嵌套一个半透明小卡片，显示“算力点数”进度条 (7,420 / 10,000)。
3. **订阅底座 (Pricing Cards)**:
   - **三栏布局**: 每栏是一个大圆角白卡片。
   - **核心焦点**: 中间卡片带“最有价值方案”浮标，边框为 Fiido 绿。
   - **价格展示**: 采用超大黑色加粗字体 (如 ¥5,999)。
4. **方案货架 (Solution Shelf)**:
   - **四栏网格**: 陈列“数据洞察”、“营销生成”、“物流监控”、“智能推荐”。
   - **状态标识**: 绿色“已激活”或灰色“可预约”。
   - **视觉特征**: 底部带有“预期 ROI”小标签，使用绿字高亮。

---

## 3. 页面快照 B：核心会话工作台 (Workspace.tsx)
**位置**：`components/Workspace.tsx`
**三栏式交互布局**：
1. **左栏 (280px)**: 会话列表。激活项带有 Fiido 绿左边框。
2. **中栏 (自适应)**: 聊天主区。
   - **气泡**: 坐席气泡为深色 (#1e293b)，客户气泡为纯白带描边。
   - **AI 灵感条**: 输入框上方有一横向滚动的“建议回复”按钮组。
3. **右栏 (340px)**: 业务集成台。
   - **Tab 切换**: 客户档案、订单集成、AI 价值收益。
   - **AI 收益看板**: 深蓝色背景卡片，显示“本月节省人工 124h”等数字。

---

## 4. 页面快照 C：效能监控大屏 (Dashboard.tsx)
**位置**：`components/Dashboard.tsx`
- **图表风格**: 采用 `Recharts` 绘制的面积图。
- **特征**: 曲线下方有 Fiido 绿的渐变阴影填充，关键点带发光特效。
- **状态卡片**: 顶部四个大数字卡片，分别显示会话总数、响应时长、满意度、质检评级。

---

## 5. 交互动态逻辑 (Motion)
- **淡入动画**: `animate-in fade-in duration-700` 用于所有主视图切换。
- **悬停特效**: 货架卡片在悬停时会产生 `shadow-2xl` 并上移 `hover:-translate-y-2`。
- **骨架屏**: `BillingView.tsx`
