# AI 智能客服 - 产品需求文档

## 产品概述

AI 智能客服是 Fiido 智能服务平台的核心产品，为用户提供 24/7 自助服务能力。

## 核心功能

### 1. 多轮对话
- 基于 Coze Workflow 的智能对话
- 支持上下文理解
- 自动意图识别

### 2. 订单查询
- 支持订单号查询
- 支持邮箱查询多订单
- 物流状态追踪

### 3. 人工转接
- 智能判断转接时机
- 班次控制
- 排队管理

## 用户场景

1. 用户查询订单状态
2. 用户咨询产品问题
3. 用户请求人工服务

## API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| /api/chat | POST | 同步聊天 |
| /api/chat/stream | POST | 流式聊天 |
| /api/conversation/* | POST | 会话管理 |
| /api/bot/info | GET | 机器人信息 |
