## 📋 模块1: 会话质检系统

### 1.1 功能概述

自动化和人工相结合的会话质检系统，确保服务质量。

**参考对标**：
- 呼叫中心质检系统
- 智齿客服质检功能

### 1.2 功能需求

#### 1.2.1 自动质检

**F1-1: 响应时间检测**
- 首次响应时间 < 30秒
- 平均响应时间 < 60秒
- 超时预警

**F1-2: 敏感词检测**
- 禁用词检测（脏话、攻击性语言）
- 不当承诺检测
- 泄密风险检测

**F1-3: 服务规范检测**
- 问候语检测
- 结束语检测
- 专业用语检测

#### 1.2.2 人工抽检

**F1-4: 抽检规则配置**
- 按比例随机抽检（如5%）
- 按坐席抽检
- 按问题类型抽检

**F1-5: 质检评分表**
包含多个维度评分：
- 服务态度（20分）
- 专业能力（30分）
- 响应速度（20分）
- 问题解决（30分）

## 📋 模块2: 坐席绩效考核

### 2.1 功能概述

多维度的坐席绩效考核体系。

### 2.2 功能需求

#### 2.2.1 KPI指标

**F2-1: 核心KPI指标**

| 指标 | 计算方式 | 目标值 |
|-----|---------|--------|
| 接待会话数 | 当日接待数量 | ≥ 20 |
| 平均响应时间 | 首次响应平均值 | ≤ 30秒 |
| 问题解决率 | 解决数/总数 | ≥ 85% |
| 客户满意度 | 5分制平均分 | ≥ 4.5 |
| 质检合格率 | 合格数/抽检数 | ≥ 90% |

**F2-2: 绩效排行榜**
- 日/周/月排行
- 多维度排行
- 进步幅度排行

**F2-3: 绩效报表**
- 个人绩效报表
- 团队绩效对比
- 趋势分析图表
