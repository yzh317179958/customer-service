# 全局指导文档

本目录包含系统的最高规格指导文档，定义整体架构、功能范围和核心目标。

## 文档说明

| 文档 | 说明 | 阅读顺序 |
|------|------|----------|
| PROJECT_README.md | 项目概述和快速入门 | 1 |
| prd.md | 主PRD - 系统需求定义和约束 | 2 |
| PRD_COMPLETE_v3.0.md | 完整版PRD - 详细功能规格 | 3 |

## 核心内容

### prd.md 主要内容
- 系统架构概述
- 核心功能定义
- 技术约束和原则
- 开发规范

### PRD_COMPLETE_v3.0.md 主要内容
- P0 核心功能详细规格
- P1 增强功能规格
- P2/P3 可选功能
- 状态机设计
- API 设计原则

## 使用建议

1. **新成员**: 先阅读 PROJECT_README.md 了解项目背景
2. **开发前**: 必读 prd.md 理解约束和原则
3. **功能开发**: 参考 PRD_COMPLETE_v3.0.md 的详细规格
