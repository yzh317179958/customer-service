## 二、产品功能类

### 2.1 这个产品是做什么的？

```
简单说，Fiido 是专门给跨境电商做的 AI 客服。

主要解决 3 个问题：
1️⃣ 物流咨询太多 → AI 自动查 17track 回复
2️⃣ 订单问题反复问 → 接入 Shopify 一键查
3️⃣ 时差睡不好 → 24 小时 AI 在线托管

实测下来，70% 的咨询 AI 能直接搞定，不用人管。

你现在客服这块主要遇到什么问题？
```

### 2.2 跟其他客服系统有什么区别？

```
最大的区别是：我们专门为跨境电商设计的。

对比一下：
┌────────────┬──────────┬──────────┐

│ 物流追踪    │ 没有     │ 17track  │
│ 多语言     │ 要加钱    │ 自带     │
│ 起步价     │ ¥1800/年 │ ¥0免费版 │
└────────────┴──────────┴──────────┘

另外我们现在团队小，服务很贴：有问题微信直接找我，不用提工单排队等。

你之前用过什么客服系统吗？
```

### 2.3 支持哪些平台？

```
目前主要支持 Shopify，可以同时接入多个站点（UK、DE、FR、US 等等都行）。

物流追踪这块对接的是 17track，支持全球 1000+ 物流商。

后续会支持 WooCommerce、独立站等，如果你有特定需求可以告诉我，我们可以优先排期。

你现在用的是 Shopify 吗？
```

### 2.4 AI 能处理哪些问题？

```
AI 主要处理这几类：

✅ 能处理：
• 订单查询（"我的订单到哪了"）
• 物流追踪（"包裹什么时候到"）
• 售前咨询（"这个产品怎么样"）
• 常见问题（退换货政策、尺码表等）

⚠️ 会转人工：
• 投诉纠纷
• 复杂售后（要退款、换货）
• AI 不确定的问题

实测 70% 左右能自动处理，剩下 30% 转人工，不会让客户干等着。
```

### 2.5 AI 回答不准怎么办？

```
这个不用担心：

1️⃣ AI 不确定时会自动转人工，不会瞎答
2️⃣ 你可以在后台调整话术、补充知识库
3️⃣ 我们可以帮你做 AI 训练，针对你的品牌定制

另外我们有个承诺：首月 AI 解决率不到 50%，全额退款。

所以放心试就行，没效果不要你钱。
```

## 九、话术原则

### 9.1 沟通风格

| 要 | 不要 |
|----|----|
| 像朋友聊天 | 机械复制话术 |
| 先问再答 | 上来就推销 |
| 承认不足 | 夸大其词 |
| 给选择 | 强迫决定 |
| 解决问题 | 逃避问题 |

### 9.2 禁忌

- ❌ 不要诋毁竞品
- ❌ 不要承诺做不到的事
- ❌ 不要催得太紧让人反感
- ❌ 不要已读不回用户消息
- ❌ 不要用"亲"等淘宝体

### 9.3 每次沟通目标

```
1. 了解需求（问清楚情况）
2. 解决疑虑（针对性回答）
3. 推进一步（试用/付费/加微信）
```
