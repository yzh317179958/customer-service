```
• AI 无法回答时，会自动转接人工客服

• 我们提供 AI 训练服务，针对你的品牌定制

首月 AI 解决率不到 50%？我们全额退款。
```

**Q4：支持哪些语言？**
```
目前支持英语和中文自动识别切换。
客户用英语问，AI 用英语答；用中文问，用中文答。
德语、法语、西班牙语版本开发中。
```

**Q5：数据安全吗？**
```
• 所有数据加密传输（HTTPS）
• 服务器部署在阿里云，通过等保认证
• 不会向第三方共享你的客户数据
• 支持企业版私有化部署
```

**Q6：可以随时取消吗？**
```
当然可以。
• 月付用户：当月结束后自动停止，不自动续费
• 年付用户：30 天内不满意全额退款
• 没有任何隐藏费用或合同绑定
```
